# Job Application Automation Agent (JAAA)

Autonomous agent designed to navigate real company career sites (ATS), handle account creation, email/OTP verification, form completion, resume upload, CAPTCHA bypass, and submission archival.

## Core Capabilities
- **Native ATS Support**: Operates on Workday, Greenhouse, Lever, iCIMS, Taleo, etc.
- **Email Intelligence**: Real-time magic link and OTP extraction (Gmail API/IMAP).
- **CAPTCHA Resolution**: Automatic bypass of reCAPTCHA, hCaptcha, and Cloudflare Turnstile via CapSolver/2Captcha.
- **AI-Driven Form Filling**: Uses Stagehand (Playwright + LLM) for resilient element detection and field mapping.
- **Proof of Work**: Captures full-page screenshots and logs all applications in PostgreSQL.

## Tech Stack
- **Browser**: Playwright + Stagehand (Browserbase managed browsers)
- **Orchestration**: Claude 3.5 Sonnet / 4.6
- **Backend**: Python (FastAPI + Celery + Redis)
- **Database**: PostgreSQL
- **CAPTCHA**: CapSolver (Primary), 2Captcha (Fallback)

## Project Structure
- `orchestrator/`: LLM planning and decision logic.
- `browser/`: Playwright + Stagehand interaction layer.
- `email_handler/`: Verification link and OTP extraction.
- `captcha/`: Resolution and token injection.
- `storage/`: DB models and screenshot management.
- `resume/`: PDF parsing and tailoring.
